## 📌 Featured Projects

### 🏦 [Credit Risk & Probability of Default (PD) Model](https://github.com/NikhilMishraDS/credit-pd-model)
> End-to-end ML framework for estimating borrower default probability — aligned with Basel III IRB and IFRS 9 ECL standards.

- XGBoost model achieving **~0.96 ROC-AUC** on hold-out test set
- **SHAP explainability** for regulatory-grade model transparency (SR 11-7)
- Risk segmentation: borrowers bucketed into 5 tiers (Very Low → Very High Risk)
- Custom feature engineering: LTI ratio, credit maturity index, income stability score
- `Python` `XGBoost` `SHAP` `Scikit-learn` `Basel III` `IFRS 9`
